# Pokedex Web 
#### using PokeApi & TailwindCSS
Browse Pokemon by Region, click to see the species detail and evolution.

## How to Run

```bash
npm install
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
